# FFMPEG-API
## Description

Built using [Nest](https://github.com/nestjs/nest) framework for TypeScript.

## Installation

Assumpions:
* NodeJS 11+ is installed locally

```bash
$ npm install
```

## Running the app

```bash
# development (recommended mode for a quick test)
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## View the swagger

Open [Swagger Documentation for localhost](http://localhost:3000/swagger) to test and play with the api.

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
